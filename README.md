# Projetos Interface Gráfica

Este espaço centraliza os estudos de caso e aplicações desenvolvidas ao longo da disciplina de **Técnicas de Desenvolvimento de Interface Gráfica**.

As instruções e requisitos para a execução destes projetos foram baseados nas diretrizes do curso **Responsive Web Desgin** da plataforma **freecodecamp.org**.

## Autor
* Allan Guilherme da Silva Vieira


## Projetos Desenvolvidos
1. **Formulário de Pesquisa**: Aplicação web destinada ao registro e armazenamento de dados dos graduandos em Computação da UEPB.
2. **Página de Tributo**: Tributo a Heath Ledger.
3. **Página de Portfólio Pessoal**: Apresenta o perfil, projetos e contato do desenvolvedor(autor).


## Linguagens e Tecnologias
* **HTML5**: Estruturação semântica do conteúdo.
* **CSS3**: Estilização e organização de layout.
* **VS Code**: Ambiente de desenvolvimento.
